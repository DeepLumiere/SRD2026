# Notebooks

This directory contains Jupyter notebooks for analysis, visualization, and demonstrations.

## Notebooks

- **demo.ipynb**: Quick demonstration of the model capabilities
- **data_exploration.ipynb**: Exploratory data analysis
- **results_analysis.ipynb**: Analysis of experimental results
- **visualization.ipynb**: Visualization of model outputs

## Usage

To run the notebooks:

```bash
# Install Jupyter if not already installed
pip install jupyter notebook

# Launch Jupyter
jupyter notebook
```

Navigate to the desired notebook and run the cells.

## Requirements

All notebooks assume you have installed the SRD2026 package and its dependencies. Some notebooks may require additional packages like:
- seaborn
- plotly
- ipywidgets

Install them with:
```bash
pip install seaborn plotly ipywidgets
```
